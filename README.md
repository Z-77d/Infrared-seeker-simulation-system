# Infrared Seeker Imaging Simulation System

A physics-based infrared seeker imaging simulation framework covering the full imaging chain from target radiation to the final rendered image.

## Overview

This system simulates the infrared imaging process through four major stages:

1. **Radiative Transfer Model**
2. **Detector Response Model**
3. **Optical Projection Model**
4. **Aero-Optical Effects**
